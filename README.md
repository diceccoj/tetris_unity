# tetris_unity


*Note: due to Github size limits, this repository only contains the project's 'Assets' folder. All the scripts and materials are present, but the whole project cannot be found here.


Controls:

S/Down: Soft Drop

A/Left: Shift Left

D/Right: Shift Right

Up/W: Hard Drop

Space: Rotate
